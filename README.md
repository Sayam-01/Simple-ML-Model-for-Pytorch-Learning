# Simple-ML-Model-for-Pytorch-Learning
This is a repo for pytorch fundamentals.

This repo will follow the Daniel Bourke's Youtube course on pytorch. 

**Course Link**: https://www.youtube.com/watch?v=Z_ikDlimN6A 
## Features (will be updated as I progress in the course):

1. The first part contains most of the PyTorch fundamentals.
   - PyTorch introduction
   - Introduction to tensors
   - Shapes of tensors
   - Important tensor functions

2. The second part contains ML fundamentals.
   - We build a very very very simple Linear Regression model.
   - Learn about data, parameters, hyperparameters
   - Learn about test sets, train sets, etc
   - Learn about training a model through loss function, optimizer, and training loops.
   - Learn about testing loops and making predcitions with the learned params. 
   - Learn about saving and loading a model in Pytorch 
   - Learn about device agnostic code

3. The third part contains NN classification problems 
   - Introduction to classification problem
   - Creating a Binary classification problem  
   - Differnce between classification and regressin problems 
   - About logits, prediction probabilites
   - Concept about Non-linearity and why its needed
   - About non-linear activation functions
   - Making a choice with the model
